    Etapas para adicionar um repositório ao GitHub
    1. Criar o repositório
    2. Copiar o link de clonagem
    3. Abrir o terminal no VScode
    4. Mover até a pasta desejada (cd + tab)
    5. Clonar o link com (git clone + ctrl+v)
      • Se der errado, basta mover o arquivo html para a pasta que foi clonada, pois o git está la dentro

    6. Mover o terminal até chegar na pastar em que está o Git (cd + tab)
    7. Ao chegar na pasta, coferir o git status
    8. git add .
    9. git commit -m""
    10. git push
